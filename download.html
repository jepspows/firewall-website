Installation

3 steps, 10 seconds

No installer wizard. No admin permissions. Just download, open, done.

1

Download

Click the button above for your platform. One file — .app for Mac, .exe for Windows.

2

Unblock (Windows)

Right-click Firewall.exe → Properties → check "Unblock" at the bottom → Apply. This removes Windows' "Mark of the Web" lock. Then double-click to run.

3

Dashboard Opens

The shield icon appears in your menu bar / system tray. Your browser opens to the Firewall dashboard automatically.

Troubleshooting: If Windows says "another program is using this file" or "Windows protected your PC" — this is normal for unsigned executables downloaded from the internet. Windows Defender is scanning the file. Wait 30 seconds after downloading, then right-click Properties → Unblock. If SmartScreen blocks it, click "More info" → "Run anyway".
